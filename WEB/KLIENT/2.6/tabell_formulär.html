Table layout

Table-layout egenskapen kan i CSS användas för att sätta vilken algoritm en tabell ska använda.
Olika värden påverkar hur tabellen ser ut på olika sätt

Exempel på olika värden

  • Auto - Denna algoritm är standardvärdet för en tabell. Auto innebär.
    att columnernas bredd sätts utifrån det bredaste innehållet i tabellen där det inte går att göra en radbrytning.
    Algoritmen kan vara långsam då innehållet i tabellen först måste läsas igenom innan den kan skapas
  • Fixed - Denna algoritm formar istället tabellen utifrån hur bred tabellen är satt att vara, istället för innehållet.
    Detta gör att tabellen snabbare kan läsas in då browsern inte behöver gå igenom innehållet.

CSS och formulär

CSS kan också användas för att ändra utseendet för HTML-formulär.
Man kan exempelvis sätta bredd, padding, färg eller border på ett input-fält,

Textfält med CSS: 


CSS: input[type=text] {
        background-color: white;
        margin: 20px;
        border: 1px solid blue;
        border-radius: 25px;
        }

CSS: input[type=password]{
        background-color: lightgray;
        margin: 20px;
        border: 1px solid black;
        }

CSS: textarea {
        background-color: lightgray;
        margin: 20px;
        height: 100px;
        width: 200px;
        resize: none;
        }

CSS: input[type=button]{
        background-color: forestgreen;
        border: 1px solid black;
        color: white;
        padding: 15px 30px;
        text-decoration: none;
        margin: 20px;
        cursor: pointer;
        }
