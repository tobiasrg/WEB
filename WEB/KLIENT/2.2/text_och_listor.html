Font-family

En fontfamilj används i CSS och specifierar vilken font ett visst element ska ha. Detta kan exempelvis skrivas på följande sätt:

p.serif{

font-family: "Times new roman", serif;

}

Man kan genom detta sedan anropa fontfamiljen i html genom att ge p-taggen class="serif".
I detta fall används fontfamiljen Times new roman, men har som nödlösning att använda den generiska familjen serif om Times new roman inte stödjs.

Man kan också ange hur stor en text ska vara i % eller em. Dessa typer av font-size är relativa. % sätter fonten till en % av det överordnade elementets storlek.
Em sätter fonten relativt till font-sizen av elementet, 2em betyder alltså att fonten är två gånger större än den ursprungliga fonten

Teckensnitt

Som tidigare nämnt finns det alltså font-familys för att ange vilken font en viss text ska ha. Inom begreppet font finns det generiska familjer, vilka exempelvis är serif, sans serif eller monospace
samt font familjer, vilka exempelvis är "Times New Roman", "Arial" eller "Helvetica". De generisa familjerna är en grupp av fontfamiljer med ett liknande utseende, medans font-familjer är specifika fonter

Genom att endast skriva font: i CSS kan man dock specifiera flera egenskaper för en text på en gång, exempelvis (i CSS) font: 20px arial, sans-serif;;

Listor

Med CSS-egenskapen list-style kan man ange olika stilar för listor i HTML. Likt CSS-egenskapen font kan man med list-style sätta alla egenskaper för en lista på en gång
