I head-taggen lagras all metadata, vilket är data om själva html-dokumentet och ligger placerad mellan html-taggen och body-taggen
Exempel på vad en metatagg kan innehålla är exempelvis en beskrivning av hemsidan(description), författaren (author), nyckelord(keywords) för sökning eller teckenuppsättning(charset).
Meta-taggen robots gör att sidan inte visas för en sökmotors sökresultat, och taggen generator används för att beskriva vilket program som användes för att bygga hemsidan.
http-equiv kan exempelvis användas för att sätta intervall för uppdatering på hemsidan. exempel på metataggar:
<meta name="keywords" content="HTML, CSS, XML, Javascript>
<meta name="description content="Grunderna i HTML 5>
<meta name="author" content="Tobias Rydberg">
<meta name="http-equiv content="30">
<meta name=robots content=">
I headern lagras också title-taggen, vilken bestämmer titeln för dokumentet. Title-taggen visar namnet på sidan i browsern, favoriter, eller när sidan visas i sökmotorer
Exempel på denna sida: <title> Grunderna i HTML5</title>
