Länktillstånd

En länk kan stylas med CSS beroende på vilket tillstånd den befinner sig i. Som standardläge brukar länkar vara blåa och understrukna när de är obesökta
men ändras till lila färg när de är besökta. Genom att tillämpa de olika länktillstånden i CSS kan man dock få detta att se annorlunda ut

Det finns fyra olika länktillstånd, vilka är:

  • a:link - en normal, obesökt länk
  • a:visited - en länk som användaren har besökt
  • a:hover - en länk när användaren håller musen över den
  • a:active - en länk i den stunden användaren klickar på denna

Använder man sig av flera av dessa tillstånd i CSS är det viktigt att de skrivs i just denna ordning för att de ska fungera

Man kan även använda sig av cursor: för att ändra musens utseende när denna pekar på elementet

EXEMPEL

Länktillstånd CSS
:link .exempel:link{color:black;}
:visited .exempel:visited { color: orange; }
:hover .exempel:hover { font-size: 2em; color: orange; }
:active .exempel:active { color: black; }
.exempel { cursor: crosshair; }
